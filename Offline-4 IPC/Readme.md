# IPC

## How to run ??

### 1. using Make

- **Compile the program**
- **Run the program**
    
    ```bash
    make run
    ```
    
- **Clean up the compiled binary**
    
    ```bash
    make clean
    ```
    

### 2. manually

- m the program
    
    ```bash
    g++ -o ipc ipc.cpp -pthread
    ```
    
- run the program
